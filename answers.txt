السؤال 1: ما الهدف من مستودع Git؟ وكيف يساعد مطوري الواجهة الأمامية في إدارة مشروع مثل صفحة ويب HTML/CSS/JS؟
حفظ نسخ من ملفات HTML وCSS وJavaScript الخاصة بك.

الرجوع إلى نسخة أقدم إذا حدث خطأ ما.

العمل مع الآخرين دون أن يفقد أحدكم تغييرات الآخر.

تجربة ميزات جديدة بأمان باستخدام الفروع (Branches).

الحفاظ على تنظيم مشروعك وسلامته.

بالنسبة لمطوري الواجهة الأمامية الذين ينشئون صفحات ويب، يجعل Git من السهل إدارة التغييرات، وإصلاح الأخطاء، والعمل ضمن فريق — وكل ذلك مع الحفاظ على الكود نظيفًا وتحت السيطرة.

السؤال 2: اشرح الفرق بين الملفات المتتبعة (Tracked) وغير المتتبعة (Untracked) في Git، ولماذا قد تسبب الملفات غير المتتبعة مشاكل في مشروع واجهة أمامية؟
الملفات المتتبعة: هي الملفات التي يعرف Git عنها مسبقًا — تم إضافتها (add) وتثبيتها (commit) من قبل.

الملفات غير المتتبعة: هي ملفات جديدة لم يرها Git بعد ولم يتم حفظها في أي عملية commit.

لماذا قد تسبب الملفات غير المتتبعة مشاكل في مشروع واجهة أمامية؟

لأنها غير محفوظة في نظام التحكم بالإصدار.

لا يمكن لأعضاء الفريق الآخرين رؤيتها.

قد تفشل عملية البناء (build) على أجهزة الآخرين.

يمكن أن تؤدي إلى ارتباك أو تكرار في العمل.

السؤال 3: لماذا من المهم استخدام ملف لتجاهل عناصر معينة (مثل node_modules) في مستودع Git، خاصة في مشاريع الواجهة الأمامية؟
تجنب التخزين غير الضروري

مجلد node_modules قد يحتوي على آلاف الملفات ويشغل مساحة كبيرة.

إذا قمت بعمل commit له، يصبح المستودع ضخمًا وبطيئًا.

الأمان

بعض المجلدات (مثل .env) قد تحتوي على معلومات حساسة مثل مفاتيح الـ API.

قد يؤدي إضافتها بالخطأ إلى كشف أسرار المشروع.

تحسين التعاون

لا يحتاج المطورون الآخرون إلى نفس الملفات المؤقتة أو ملفات البناء — يمكنهم إنشاؤها محليًا.

هذا يحافظ على نظافة المستودع وتركيزه على الكود الفعلي.

السؤال 4: كيف يستفيد مطور الواجهة الأمامية من عرض سجل المشروع أثناء العمل على نسخ متعددة من صفحة ويب؟
رؤية التعديلات التي تمت.

معرفة من قام بها ومتى.

الرجوع إلى نسخة تعمل إذا حدث خلل.

السؤال 5: ما دور GitHub بالنسبة إلى مستودع Git المحلي، وكيف يضيف استضافة المشروع على GitHub Pages قيمة لمطور الواجهة الأمامية؟
GitHub يربط الكود المحلي الخاص بك بالعالم — من أجل النسخ الاحتياطي، والتعاون، والمشاركة.

GitHub Pages يوفر لمطوري الواجهة الأمامية استضافة مجانية وفورية — مما يسهل عرض المشاريع للعملاء أو أصحاب العمل أو الأصدقاء.
